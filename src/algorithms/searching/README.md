 ## 1. 순차 탐색 (sequential search), 선형 탐색 (linear search)
 
+ 가장 간단하고 직접적인 탐색 방법
+ 배열의 요소들을 처음부터 마지막까지 하나씩 검사하여 값을 찾는다
+ 구현이 간단하지만 **비효율적**이다
