# 우선순위 큐(priority queue)

+ 보통의 큐처럼 먼저 들어온 데이터가 먼저 나가는 게(FIFO) 아닌 **우선순위가 높은 데이터가 먼저 출력되는 구조**이다.
+ 배열, 연결 리스트 등 여러 가지 방법으로 구현이 가능하지만 가장 효율적인 자료구조는 **힙(heap)** 이다!
