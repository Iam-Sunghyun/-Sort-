## 4. 셸 정렬 (shell sort)

### 셸 정렬이란?
 가장 오래된 정렬 알고리즘 중 하나로, 1959년 'Donald L. Shell'이란 사람이 제안한 알고리즘    
 삽입 정렬이 어느정도 정렬된 배열에 대해서는 매우 빠른 것에 착안하여 만들었다
